# Missed Day Language Rules (Phase 5H)

## Why this document exists

Sabil must never convert spiritual return into shame.

This document defines banned and approved language when users return after gaps.

## Hard Prohibitions

Never use phrases equivalent to:
- "You missed X days"
- "You are behind"
- "Catch up now"
- "Don't break your streak"
- "You failed to stay consistent"

Never imply:
- spiritual worth is reduced by pauses
- user owes the app output
- urgency is the only faithful response

## Required Framing

Return messages must:
- begin with welcome
- normalize pause without moral panic
- offer one small next step
- end with gentle reassurance

## Approved English Patterns

- "Welcome back. Your place is still here."
- "No rush. Begin from where your heart is today."
- "One calm page is enough for now."

## Approved Urdu Patterns

- "خوش آمدید۔ آپ کی جگہ اب بھی یہیں ہے۔"
- "جلدی نہیں۔ دل جہاں آمادہ ہو وہیں سے شروع کریں۔"
- "ابھی کے لیے ایک پُرسکون صفحہ کافی ہے۔"

## QA Checklist (Return Copy)

Before merge, verify:
1. No direct/indirect "behind" language.
2. No hidden urgency verbs.
3. No guilt-loaded religious wording.
4. No productivity metaphors.
5. Emotional safety preserved in both EN/UR.

If any check fails, copy must be rewritten.
